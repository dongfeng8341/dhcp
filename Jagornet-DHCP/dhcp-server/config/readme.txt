Jagornet DHCP Server configuration files
========================================

dhcpserver.xml - default Jagornet DHCP server configuration file
dhcpserver-basic.xml - basic example Jagornet DHCP server configuration file
dhcpserver-sample.xml - complex example Jagornet DHCP server configuration file
dhcpserver.xsd - Jagornet DHCP server configuration XML Schema file
log4j.properties - Jagornet DHCP server Log4J configuration file
jdbc.properties - placeholder for experimental JDBC configuration
